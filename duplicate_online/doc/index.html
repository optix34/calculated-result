Список объектов клиента – расширение PILOT

Версия 1.0

Добавляет в левую панель навигации вкладку «Список объектов», где отображаются все транспортные средства клиента со всеми полями, возвращаемыми API /ax/tree.php.

Функции

  • Динамическое создание колонок на основе полей объектов.
  • Фильтрация по состоянию (Все, Активные, Аварии, Стоянка, Холостой ход).
  • Поиск по названию объекта.
  • Правая область разделена на две горизонтальные панели (пустые).

Установка

  1. Разместите файлы расширения на хостинге (например, Cloudflare Pages).
  2. Убедитесь, что Module.js доступен по прямому URL.
  3. В административной панели PILOT добавьте приложение с именем client_objects_list и базовым URL, указывающим на папку с расширением.
  4. Включите расширение для пользователя и перезагрузите PILOT.

Примечания

Если какие-либо поля отсутствуют, отображается прочерк. Для статуса используются иконки FontAwesome (должна быть загружена в PILOT).
