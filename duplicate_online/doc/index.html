Расширение PILOT: Дубликат Онлайн

Версия: 1.0

Данное расширение добавляет в левую навигацию PILOT новую вкладку «Дубликат Онлайн», которая полностью повторяет функционал стандартной вкладки «Онлайн»:

  • Дерево групп и транспортных средств, загружаемое через /ax/tree.php.
  • Фильтрация по состоянию: Все, Активные, Аварии, Стоянка, Холостой ход.
  • Поиск по имени объекта.
  • Раскрытие/сворачивание групп.

Правая область разделена по горизонтали на две пустые панели (может быть использована для будущих доработок).

Установка

  1. Разместите файлы расширения на любом статическом хостинге (Cloudflare Pages, GitHub Pages, ваш сервер).
  2. Убедитесь, что Module.js доступен по прямому URL, например https://ваш-хост/duplicate_online/Module.js.
  3. В панели администратора PILOT добавьте новое приложение:
    • Название: Дубликат Онлайн
    • URL: https://ваш-хост/duplicate_online/ (слеш в конце)
    • Класс: Store.duplicate_online.Module
  4. Включите расширение для нужных пользователей и перезагрузите PILOT.

Примечания

Расширение не модифицирует оригинальную вкладку «Онлайн» и не требует дополнительных библиотек.

Для корректной работы фильтров по состоянию API /ax/tree.php должен поддерживать параметр state (1 – все, 2 – аварии, 3 – стоянка, 4 – холостой ход). Если поведение отличается, можно адаптировать код в filterByState.
